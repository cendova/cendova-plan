# Sicherheitshinweise

CendovaPlan ist ein reines Frontend ohne Backend: Es gibt keinen Server,
keine Konten und keine Datenübertragung — DICOM-Bilder und Pläne bleiben
im Browser bzw. auf dem Rechner der Nutzerin/des Nutzers.

## Schwachstelle melden

Bitte **nicht** als öffentliches Issue, sondern vertraulich über
GitHub Security Advisories melden: Repository → Tab **Security** →
**Report a vulnerability**. Wir melden uns so schnell wie möglich zurück.

## Unterstützte Version

Es gibt keine gepflegten Release-Zweige — maßgeblich ist der aktuelle
Stand von `main`.
